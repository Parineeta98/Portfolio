Research Projects

Simulating Visual Imagination in a Memory Model

Explored how the brain generates images using a simulation model of memory in monkeys. Added a new brain region to simulate communication between prefrontal and visual areas, helping explain how different brain regions coordinate during visual imagination. The model uses real brain connectivity data for biological realism.

Wong-Wang working memory model · Python · NumPy · Matplotlib · Pandas

Visual Search Task Experiment

Implemented a Visual Search Task (VST) based on Feature Integration Theory (FIT) to study how people identify targets in visual environments. Participants searched for an odd symbol among distractors across three conditions — disjunctive by color, disjunctive by shape, and conjunctive search — plus a control task.

Python · NumPy · Pandas · Matplotlib

Shape Flanker Task Experiment

Explored selective attention and inhibitory control using a modified Flanker Task, testing whether the Flanker Effect (slower responses when surrounded by conflicting stimuli) holds for shapes the way it does for arrows or letters. Participants responded to target shapes while ignoring surrounding distractors.

Python · NumPy · Pandas · Matplotlib · Object-Oriented Design

EEG Study — Visual Processing Speed

Studied how fast people process visual information when making decisions, using EEG to observe brain activity around 200ms post-stimulus. Signal components were estimated via Singular Value Decomposition (SVD) and analyzed using linear and Bayesian regression models.

Python · MNE · Statsmodels · JASP · ICA · SVD

Neuron Simulation Study

Used computer simulations to understand how communication speed between neurons shapes their collective behavior. Built a virtual model of two connected neurons and varied their communication delay to observe changes in firing patterns.

Python · NumPy · SciPy · Brian Simulator · Pinsky-Rinzel & Wang-Buzsáki models
